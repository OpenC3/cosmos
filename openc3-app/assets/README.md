# openc3-app assets

## openc3-icons.ttf

A tiny icon font (family `openc3-icons`) containing the app's UI glyphs, embedded
in the binary via `include_bytes!` and loaded with iced's `.font(...)`. We use an
embedded font rather than a Unicode glyph or an svg/canvas widget because:

- the bundled UI font has no gear glyph (U+2699 renders as tofu), and
- iced's tiny-skia software renderer (this app avoids the wgpu/GPU backend) does
  not reliably paint the svg or canvas geometry layers.

Text, however, renders and positions correctly on every platform, so an embedded
font is the portable way to get vector icons.

### Glyphs

| Codepoint | Name | Use            |
|-----------|------|----------------|
| U+E900    | gear | Settings button |

### Regenerating

The font is generated by `tools/gen_icons.py` (needs `fonttools`):

```bash
cd openc3-app
uv run --with fonttools python tools/gen_icons.py
```

Commit the regenerated `assets/openc3-icons.ttf`.
