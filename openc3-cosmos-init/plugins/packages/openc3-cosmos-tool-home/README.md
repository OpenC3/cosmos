## OpenC3 COSMOS Home Plugin

[Documentation](https://openc3.com)

This plugin provides the COSMOS welcome landing page with getting-started links and resources.

## Getting Started

1.  At the OpenC3 Admin - Plugins, upload the openc3-cosmos-tool-home.gem file
