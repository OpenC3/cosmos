Tools

🧑‍⚖️ Admin

Administer COSMOS, install plugins, change settings

🤖 Autonomic (Enterprise)

Automated execution of commands and scripts

🪣 Bucket Explorer

File browser to the COSMOS backend bucket storage system

🗓️ Calendar (Enterprise)

Calendar visualization of metadata, notes, and timelines

📢 Command Sender

Send individual commands

📡 Command and Telemetry Server

Status about interfaces, targets and log messages

📋 Command History (Enterprise)

See all the commands sent, by whom, and if successful

📥 Command Queue (Enterprise)

See all the command queues and interact with commands

🛠️ Data Extractor

Extract command or telemetry data into files

🖥️ Data Viewer

View packet or item data

📖 Handbooks

Format the command and telemetry definition into a webpage

🚥 Limits Monitor

View out of limit items and log messages

🔭 Log Explorer (Enterprise)

View, search, and export historical logs

🖥️ Packet Viewer

Displays all packets with their items

🏃‍➡️ Script Runner

Run Python or Ruby scripts to send commands and check telemetry

💚 System Health (Enterprise)

Visualization of overall System Health metrics

🪑 Table Manager

Binary file editor with upload / download

📈 Telemetry Grapher

Graph real time or historical data

🖥️ Telemetry Viewer

Build custom screens using built-in widgets
